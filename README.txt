Suede
=====
Windows API keylogger

Install
=======
x86_64-w64-mingw32-gcc -mwindows -osuede suede.c

Or for extra checks

x86_64-w64-mingw32-gcc -Wall -Wextra -pedantic -std=c99 \
  -mwindows -osuede suede.c
